A premium, highly-interactive, responsive single-page portfolio website showcasing Flutter and Full-Stack development skills, projects, and professional experience. Built with clean HTML, CSS variables, custom cursors, dynamic hover card glows, and smooth scroll animations.

---

## 🚀 Live Demo & Deployment on Render

This portfolio is fully optimized and configured for **Render** as a **Static Site**. Render offers a global CDN, automatic SSL certificates, and auto-deployments directly from your GitHub repository.

### How to Deploy this Portfolio on Render:

1. **Sign in to Render:**
   - Go to [render.com](https://render.com/) and sign in with your GitHub account.

2. **Create a New Static Site:**
   - Click on the **"New +"** button in the dashboard and select **"Static Site"**.

3. **Connect Your Repository:**
   - Find and connect your **`Portfolio-Website`** repository from the list of GitHub repositories.

4. **Configure the Deployment Settings:**
   - **Name:** Choose a unique name for your project (e.g., `prathmesh-ahire-portfolio`).
   - **Branch:** `main`
   - **Build Command:** *Leave blank* (None)
   - **Publish Directory:** `.` (This refers to the root directory, which contains our `index.html`).

5. **Deploy:**
   - Click **"Create Static Site"**.
   - Render will build and deploy your site in a few seconds! Your portfolio will be live at a custom URL like `https://your-app-name.onrender.com`.
